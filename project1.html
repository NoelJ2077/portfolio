My first public Portfolio Site

During the second year of my apprenticeship to the Applikationsentwickler, I had some free time and i wanted to practice my Web Development skills. So i learned about a Portfolio Website and i think everybody in today's world should have a Portfolio Website. I coded my first Portfolio with a template from StartBootstrap's Resume Template. The first few hours where the hardest because i had to read template documentation and how the styling works. After that, i was able to customize the template to my liking and adding my desired Resume files and Projects.

Archieved as from Mai 2024

This Website was active on GitHub Pages from October 2023 to Mai 2024. I took it down and removed the Repository because the website was a little bit laggy and i wanted to make a new one with a better design and more features.
